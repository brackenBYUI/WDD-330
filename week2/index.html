Week 2 Notes

Reading Notes

Best practice to write each statement on a new line and terminate by a semi-colon.
There are large list of Reserved words that can not be used when naming variables.
There are six primitive data type: String, Symbol, Number, Boolean, Undefined, Null. Everything else is an object.
JavaScript has a special operator called typeof for finding out the type of a value.
Any variable declared outside of a block is global scope. It is not best practice to have global scope variables.
If let or const are not use than the variable will have a global scope.
You can use the length property to find how many items are in an array.
The join method is used to merge all items in an array into a string. a variable can be added like & that will be input between each variable.
Sets is a data structure that represents a collection of unique values, so it cannot include any duplicate values.
Sets offer a useful way to keep track of data without having to check if any values have been duplicated.
Maps are a convenient way of keeping a list of key and value pairs.
Ternary Operator is a shorthand way of writing an if else statement.

Arrow Functions

  • They are much less verbose than normal function declarations.
  • Single parameters don't need putting into parentheses
  • The body of the function doesn't need placing inside a block if it's only one line
  • The return keyword isn't required if the return statement is the only statement in the body of the function
  • They don't bind their own value of this to the function
