Object Methods: this

Arrow Functions do not have their own this. If we reference this from an arrow function its' taken from the outer function.
The value of this is defined at run-time.

Chapter 5: Objects

An object is a self-contained et of related values and functions.
Objects can inherit properties from other objects.
Object literal is an object that is created directly in the language by wrapping all its properties and methods with curly braces.
Each property is a key-value pair, separated by commas.
If the property name doesn't follow the variable naming rules it will need to be quoted.
Use the dot notation to access properties of an object.
To add a new property to an object you can simply assign a value to the new property.
Use the delete operator to remove any property.
You can even next objects. You can mix different notations to access properties.

Chapter 6: Document Object Model

The DOM represents an HTMl document as a network of connected nodes that form a tree-like structure.
The DOM treats everything on the page as a node.
There are DOM shortcuts Document.body, Document.images, Document.links, Document.anchors, Document.forms
You can get elements many different ways getElementById(), getElementsByTagName(), getElementsByClassName(), querySelector(), querySelectorAll()
Find the value of a node .firstChild, .nodeValue, .textContent
You can get and set element's attributes getAttributes(), setAttribute()
You can create Nodes createElement(), createTextNode()
After creating you will need to append the Nodes appendChild() and this will get the elements on the page.

Chapter 7: Events

Event listeners are added to elements on the page and are part of the DOM. Until the event happens the program will continue to run.
The click even occurs when a user clicks with the mouse, presses the Enter key, or taps the screen.
The recommended way of adding event listeners is on the DOM. document.body.addEventListener('click', doSomething)
You can find the coordinates of where the event occurred screenX, screenY, clientX, clientY, pageX, pageY
There are a lot of event types you can narrow them down to Mouse events, Keyboard Events, Modifier Keys, Touch Events
